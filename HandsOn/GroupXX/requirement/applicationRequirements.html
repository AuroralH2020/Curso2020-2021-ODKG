Application Summary

Grupo 08

Link Dataset

The principal idea for this application is to show a map with the locations of the bars with terrace open in a given time and allow bookings for a number of persons taking into account the number of chairs.

Application Requirements

  • Treat the CSV data
  • Show the near bars
  • Show the locations of bars
  • Search the bars with filters
  • Check the availability of terraces
  • Booking terrace
